Made with cursor.A web page for an appliance musical instrument that can simulate ELECTRONICOS FANTASTICOS: The Electric Fan Harp. It supports editing the pitch (number of holes, rotational speed), tone (shape of the holes), and can also adjust the voltage. You can see the real-time square wave shape and the corresponding pitch, as well as various basic parameters. At the same time, it can reverse calculate other parameters based on the target pitch. You can also try playing on the web page. The original intention is to make it easier for enthusiasts to design their own fan blades.

使用cursor制作。一个可以模拟ELECTRONICOS FANTASTICOS的家电乐器：Electric Fan Harp的网页，它支持编辑音高（孔数、转速）、音色（孔的形状），同时可以对电压进行调节。你能够看到实时的方波形状以及对应的音高，以及各项基本参数，同时它也能够根据目标音高进行反推得到其他参数。你还可以在网页上进行试奏，设计初衷是让爱好者可以更方便地设计自己的扇叶。
